# Anotações do Laboratório

## Principais conceitos

- Step Functions permite criar workflows visuais.
- Cada workflow é chamado de state machine.
- Cada etapa do fluxo é chamada de estado.
- O estado Choice permite criar decisões condicionais.
- A Amazon States Language usa JSON para descrever o fluxo.

## Dificuldades encontradas

- Entender a diferença entre os tipos de estados.
- Compreender como o input JSON percorre a execução.
- Organizar a lógica condicional dentro do estado Choice.

## Insights

- Step Functions é útil quando precisamos coordenar várias etapas de um processo.
- A visualização do fluxo facilita o entendimento da regra de negócio.
- A documentação do workflow é tão importante quanto a criação da máquina de estados.
